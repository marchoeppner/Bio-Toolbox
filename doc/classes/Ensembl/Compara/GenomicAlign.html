fetch_noncoding()

DESCRIPTION

Retrieves all non-coding nucleotides located within protein-coding genes from the underlying DNA fragment NOTE! non-coding DNA is different from introns. Introns are gaps between exons but may contain coding-sequence in case of alternative splicing. This method iterates over all introns and creates a minimal set (nucleotides not covered by exons in any of the transcript!)
